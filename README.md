# WidgetPaybackPL
## 1. Opis
Prosty widget pokazujący ilość punktów na karcie payback. Wystarczy wpisać numer karty payback i przeciągnąć widget na ekran główny telefonu. Widżet odświeża się samoczynnie co pół godziny. Aby szybko odświeżyć ilość punktów kliknij na widget.

## 2. Użyte biblioteki
[OKHttp](http://square.github.io/okhttp/)  
[Gson](https://github.com/google/gson)

## 3. Do pobrania
[Google Play](https://play.google.com/store/apps/details?id=pl.edu.uksw.paybackwidget)
